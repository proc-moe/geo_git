# geo_git
> A version control tools aiming for geometry shapes.
> With fancy GUIs.

# Releases
## Alpha
### Shapes
- Rectangles

### Features
- Basic Rectangle Operations
  - Create
  - Move
  - Delete
  - Differences between versions (x)
- GUI
  - show all shapes, based on egui.
  - show differences between shapes.
